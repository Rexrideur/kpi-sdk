## Installation

Install the SDK with npm in your current app.

```bash
  npm install https://github.com/TheoSigaud/kpi-sdk.git
```


## Usage

```javascript
import { useAnalyticsPage, useAnalyticsClick } from 'kpi-sdk-esgi'

function App() {

    useAnalyticsPage();
    useAnalyticsClick();

    return <Component />
}
```
